# Node.js Clean Architecture - Aula 01

Este repositório contém o código-fonte e os materiais da Aula 01 sobre Clean Architecture em Node.js.

## Conteúdo da Aula

Nesta aula, abordamos os seguintes tópicos:

- Introdução à Clean Architecture
- Princípios da Clean Architecture (SRP, OCP, LSP, ISP, DIP)
- Camadas da Clean Architecture (Entidades, Casos de Uso, Adaptadores, Frameworks e Drivers)
- Estrutura de pastas e organização do projeto
- Implementação de um exemplo simples seguindo a Clean Architecture

## Pré-requisitos

Para acompanhar esta aula, você precisará ter instalado em sua máquina:

- Node.js (versão 18 ou superior)
- npm ou yarn (gerenciador de pacotes)
- Git

## Como Rodar o Projeto

Siga os passos abaixo para clonar o repositório e rodar o projeto:

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/seu-usuario/node-clean-arch.git
   cd node-clean-arch/aula-01
   ```

2. **Instale as dependências:**

   ```bash
   npm install
   # ou
   yarn install
   ```

3. **Execute o projeto:**

   ```bash
   npm start
   # ou
   yarn start
   ```

   Isso iniciará o servidor Node.js. Você pode testar as rotas da aplicação usando ferramentas como Postman, Insomnia ou cURL.
